# Contribuer à Lichen Cognitive Kernels

Merci de l'intérêt que vous portez à l'expansion de l'univers cognitif de **Lichen-Collectives**.

## Philosophie de Contribution
Ce projet est open-source. Nous croyons en la décentralisation de l'intelligence. Tout le monde peut **forker** ce dépôt, l'adapter et l'utiliser dans ses propres applications.

Cependant, nous demandons le respect strict de l'**éthique de recherche** établie par le projet :
1.  **Pas de Malveillance** : Les personas ne doivent pas être modifiés pour générer du contenu nuisible, haineux ou destructeur.
2.  **Attribution** : Si vous utilisez ces noyaux dans un produit commercial ou public, une mention de l'auteur original (Bryan Ouellette) est requise.
3.  **Intégrité** : Respectez l'intention originale de chaque persona. Si vous modifiez drastiquement la "personnalité", créez un nouveau fichier plutôt que d'écraser l'existant.

## Comment Proposer un Changement
1.  Forkez le projet.
2.  Créez votre branche de fonctionnalité (`git checkout -b feature/AmazingPersona`).
3.  Commitez vos changements (`git commit -m 'Add some AmazingPersona'`).
4.  Poussez vers la branche (`git push origin feature/AmazingPersona`).
5.  Ouvrez une Pull Request.

Nous sommes particulièrement intéressés par des contributions optimisant la syntaxe mathématique au sein des prompts ou réduisant la consommation de tokens.
